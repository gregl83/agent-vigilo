# Agent Vigilo

Distributed evaluation and deployment gating for Generative AI systems.

> [!NOTE]
> Agent Vigilo is a focused, systems-oriented project that demonstrates how generative AI evaluation can be treated as a first-class, distributed system, covering execution, normalization, aggregation, and deployment gating.
